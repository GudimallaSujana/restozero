# RestoZero - Zero Waste Kitchen Assistant

Hackathon-grade full-stack web app to reduce restaurant food waste using AI-inspired prediction, explainable insights, gamification, and donation workflows.

## Tech Stack

- Frontend: React (Vite), Tailwind CSS, Framer Motion, Recharts
- Backend: Node.js, Express
- Database: MongoDB + Mongoose
- Integrations: OpenWeatherMap API, Nager public holiday API

## Project Structure

- `client/` - React frontend
- `server/` - Express backend
  - `models/`, `routes/`, `controllers/`, `utils/`, `middleware/`, `seed/`

## Quick Start

### 1) Backend

```bash
cd server
cp .env.example .env
npm install
npm run seed
npm run dev
```

### 2) Frontend

```bash
cd client
cp .env.example .env
npm install
npm run dev
```

Frontend runs on `http://localhost:5173`, backend on `http://localhost:5000`.
